# Listing Image Prompt Template

Use this template to generate product listing hero images for the marketplace app.

## Template

```
Flat digital illustration of a [ITEM NAME] sitting on a wooden table surface next to a smartphone. The smartphone screen shows a product listing for the same item with the text "[ITEM NAME]" and "$[PRICE]". Warm muted color palette using coral, tan, cream, and brown tones. Solid warm-toned background. Soft minimal style with rounded shapes, no outlines, subtle shadows. Cozy marketplace listing aesthetic. Simple, clean composition.
```

## Usage

Replace:
- `[ITEM NAME]` — the product name (appears twice: on table and on phone screen)
- `[PRICE]` — the listing price
